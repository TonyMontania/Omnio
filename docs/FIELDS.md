# Fields reference · Referencia de campos

The Omnio UI is in English. This document explains what each field means in **English** and **Español**, grouped by category.

La interfaz de Omnio está en inglés. Este documento explica qué significa cada campo en **inglés** y **español**, agrupado por categoría.

---

## Common fields · Campos comunes

Present in almost every category. Presentes en casi todas las categorías.

| Field | Meaning (EN) | Significado (ES) |
|---|---|---|
| **Title** | Name of the item as displayed. | Nombre del ítem tal como se muestra. |
| **Cover** | Cover image, uploaded from disk. | Imagen de portada, subida desde tu disco. |
| **Alternative titles** | Regional / original / other-language names. | Nombres regionales / originales / en otros idiomas. |
| **Description** | Short synopsis or overview. | Sinopsis o resumen breve. |
| **Notes** | Free-form personal notes, supports mini markdown (bold, italic, bullets). | Notas personales en formato libre; acepta mini markdown (negrita, cursiva, viñetas). |
| **Tags** | Free tags for filtering. | Etiquetas libres para filtrar. |
| **Rating** | Personal 0.5–5.0 star rating (half-star steps). | Puntaje personal de 0.5 a 5.0 estrellas (medio punto). |
| **Franchise** | Franchise name — items sharing the same franchise auto-appear together in the timeline. | Nombre de la franquicia — los ítems que comparten franquicia aparecen juntos en el timeline automáticamente. |
| **Related items** | Manually linked items in the same category, with a relation kind (sequel, prequel, side story, spin-off, alt version, adaptation, other). | Ítems del mismo tipo linkeados manualmente, con tipo de relación (secuela, precuela, side story, spin-off, versión alternativa, adaptación, otro). |
| **Recommendations** | Manually linked items you'd recommend based on this one. | Ítems que recomendarías si te gustó este. |
| **Review** | Long-form review, with a "contains spoilers" toggle. | Reseña larga, con toggle "contiene spoilers". |
| **Age rating (ESRB)** | E · E10+ · T · M · AO · RP. | Clasificación por edad. |

## Games · Videojuegos

| Field | Meaning (EN) | Significado (ES) |
|---|---|---|
| **Developers** | Studios or people who made the game. Multiple allowed. | Estudios o personas que hicieron el juego. Se pueden agregar varios. |
| **Publishers** | Companies that published/distributed it. Multiple allowed. | Compañías que publicaron/distribuyeron el juego. Se pueden agregar varias. |
| **Release date** | Official release date. | Fecha de lanzamiento oficial. |
| **Platforms** | Where you own/play it (PC, PS5, Switch, custom). Free-form with autocomplete. | Dónde lo tenés/jugás (PC, PS5, Switch, custom). Libre con autocompletado. |
| **Ownership** | Owned / Shared / Subscription / Unlicensed. | Propio / Compartido / Suscripción / No licenciado. |
| **Status** | Backlog / Playing / Played / Completed / Dropped. | Pendiente / Jugando / Jugado / Completado / Abandonado. |
| **Time played** | Hours and minutes in "H.MM" format (e.g. `22.49` = 22 h 49 min). | Horas y minutos en formato "H.MM" (ej. `22.49` = 22 h 49 min). |
| **Achievements unlocked / total** | Progress on achievements/trophies. | Progreso de logros/trofeos. |
| **DLC & Expansions** | List of DLC with independent status per entry. | Lista de DLC con estado independiente por entrada. |
| **Addons** | Extra add-ons or packs (cosmetics, small extras). | Add-ons o packs (cosméticos, extras chicos). |
| **Genres** | Genre tags. | Géneros. |
| **Source** | Original / Remake / Remaster / Port / Sequel / Spin-off / Other. | Original / Remake / Remaster / Port / Secuela / Spin-off / Otro. |
| **Banner image / Logo image** | Optional banner and transparent logo used in the detail view header. | Banner y logo transparente opcionales que se muestran en la cabecera de la ficha. |
| **Completion date** | Date you finished the game. Feeds the "Completed per month" chart in Insights. | Fecha en la que terminaste el juego. Alimenta el gráfico "Completados por mes" en Estadísticas. |
| **Replay history** | Log of replays with date, rating and notes per session. | Log de replays con fecha, puntaje y notas por sesión. |

## Music · Música

| Field | Meaning (EN) | Significado (ES) |
|---|---|---|
| **Type** | Single / EP / Album / OST / Live / Compilation. Album-like types show a tracklist. | Single / EP / Álbum / OST / En vivo / Recopilación. Los tipos "álbum" muestran tracklist. |
| **Artist** | Main artist name. | Nombre del artista principal. |
| **Release year** | Year the release came out. | Año de salida. |
| **Label** | Record label. | Sello discográfico. |
| **Producers** | People credited on production. | Personas acreditadas en la producción. |
| **Genres** | Genre tags. | Géneros. |
| **Tracklist** | Numbered tracks with per-track name, artist, duration, favorite ★, personal rating, listened ✓, and lyrics button. | Tracks numerados con nombre, artista, duración, favorito ★, rating personal, escuchado ✓ y botón de lyrics. |
| **Lyrics (per track)** | Modal to save/read the lyrics of an individual track. | Modal para guardar/leer la letra de un track. |
| **Part of album** | If this is a single that belongs to an album — free text for now. | Si es un single que pertenece a un álbum — texto libre. |
| **Source** | Original / Compilation / Soundtrack / Remaster / Deluxe / Reissue / Other. | Original / Recopilación / Soundtrack / Remaster / Deluxe / Reissue / Otro. |
| **Listened** | Whether you've listened to the whole thing. | Si lo escuchaste completo. |
| **Listen history** | Log of listens (like replays for games). | Log de escuchas (como los replays de juegos). |

### Artist profile · Perfil de artista

| Field | Meaning (EN) | Significado (ES) |
|---|---|---|
| **Photo / Banner** | Circular avatar + full-width banner in Spotify-like profile view. | Avatar circular + banner ancho en el perfil estilo Spotify. |
| **Discography** | Auto-generated list of releases where `artist` matches the artist name. | Lista auto-generada de releases donde `artist` coincida con el nombre. |

## Series · Series

| Field | Meaning (EN) | Significado (ES) |
|---|---|---|
| **Status** | Plan to watch / Watching / Completed / Paused / Dropped. | Por ver / Viendo / Completada / Pausada / Abandonada. |
| **Format** | Ongoing / Ended / Miniseries / Limited / Anthology / Other. | En emisión / Finalizada / Miniserie / Limitada / Antología / Otro. |
| **Cast · Directors · Showrunners · Writers** | Credits, all free-form arrays. | Elenco, directores, showrunners, guionistas — arrays libres. |
| **Network** | TV network / streaming platform. | Cadena / plataforma de streaming. |
| **Country · Language** | Country of origin, primary language. | País de origen, idioma principal. |
| **Content rating** | Free-text age rating. | Clasificación por edad, texto libre. |
| **Seasons** | Nested list. Each season: number, title, year, total episodes, watched, rating, notes. Can optionally contain a per-episode list. | Lista anidada. Cada temporada: número, título, año, episodios totales, vista, puntaje, notas. Puede contener una lista de episodios por dentro. |
| **Episodes (per season)** | Number, title, watched date, rating, notes, filler flag. | Número, título, fecha vista, puntaje, notas, flag "filler". |
| **Episodes watched / total** | Aggregated across seasons. | Suma agregada de las temporadas. |
| **Episode duration** | Average minutes per episode — used for "Total watch time" in Insights. | Minutos promedio por episodio — se usa para "Tiempo total visto". |
| **Rewatch history** | Same shape as movies. | Igual que en películas. |

## Anime & Donghua

Two independent sub-libraries under one collapsible sidebar group. Same field set.

Dos sub-bibliotecas independientes bajo un grupo colapsable en la sidebar. Mismos campos.

| Field | Meaning (EN) | Significado (ES) |
|---|---|---|
| **Studios** | Animation studios. | Estudios de animación. |
| **Format** | TV / Movie / OVA / ONA / Special / Music Video. | TV / Película / OVA / ONA / Especial / Music Video. |
| **Airing status** | Airing / Finished / Not yet aired / Cancelled. | En emisión / Finalizado / Aún no emitido / Cancelado. |
| **Watch status** | Plan to watch / Watching / Completed / Paused / Dropped. | Por ver / Viendo / Completado / Pausado / Abandonado. |
| **Season + year** | Winter / Spring / Summer / Fall of a given year. | Invierno / Primavera / Verano / Otoño de un año. |
| **Demographic** | Shonen / Shojo / Seinen / Josei. | Género demográfico. |
| **Source** | Manga / Light novel / Web novel / Novel / Original / Game / Visual novel / Other. | Manga / Light novel / Web novel / Novela / Original / Juego / Visual novel / Otro. |
| **Episode duration** | Average minutes per ep — feeds total runtime. | Duración promedio por ep — alimenta el runtime total. |
| **Aired from / to** | Air date range. | Rango de fechas de emisión. |
| **Episodes watched / total** | Progress. | Progreso. |
| **Episodes (list)** | Optional per-episode list with title, watched, rating, notes, filler flag. | Lista opcional por episodio con título, visto, puntaje, notas y flag filler. |
| **Favorite episode / note** | Pick one favorite ep and write why. | Elegí un episodio favorito y por qué. |
| **Dropped-at / reason** | If status is Dropped, at which episode and why. | Si el estado es Abandonado, en qué episodio y por qué. |

## Cross-cutting concepts · Conceptos transversales

| Concept | EN | ES |
|---|---|---|
| **Groups** | User-created collections inside a category, drag-to-reorder. | Colecciones propias del usuario dentro de cada categoría, con reordenamiento manual. |
| **Board views** | Per-status pages accessible from the sidebar under "Collections". | Vistas por estado, accesibles desde la sidebar bajo "Collections". |
| **Custom order** | Sort mode `custom` uses the drag order. Persistent per category and per group. | El modo de orden `custom` usa el orden por arrastre. Persistente por categoría y por grupo. |
| **Insights / Statistics** | Charts and top-N lists per category. | Gráficos y listas top-N por categoría. |
| **Franchise timeline** | Auto-derived scrollable row for items that share the same `franchise` field. | Fila scrolleable auto-derivada para ítems que comparten el campo `franchise`. |
| **Local-first** | Everything lives in `data.json` + `assets/` next to the app. No accounts, no telemetry, no cloud. | Todo vive en `data.json` + `assets/` al lado de la app. Sin cuentas, sin telemetría, sin nube. |
